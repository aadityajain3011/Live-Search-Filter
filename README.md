# Live Search Filter

## Definition
This project focuses on building a dynamic and interactive search feature using core web technologies. It allows users to filter visible data instantly based on their input.

## Problem Statement
Create a web page that displays a list of items along with a search input box. As the user types, the list updates dynamically to show only matching items.

## Features
- Display a list of items
- Search input field
- Real-time filtering of data
- Case-insensitive search
- No page reload required
- Shows "No results found" if no match is found

## Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation)
